My goal for BOLT is to have a simple, easy to use and learn language that anyone can pick up almost instantly.
It can't do much, but it's very good for learning.
To keep the 'easy to use and learn' aspect of this afloat, I try and keep a list of all commands updated and avaiable for anyone to look at, and see what a command does.
Hope this helps!

PRINT

Usage: PRINT "[message]"

A simple command that outputs a message to the console.

GOTO

Usage: GOTO [lineNo]

Stop executing the program to go to a specific line number.

LET

Usage: LET [variableName] = [value]

Initalize a variable, and set it to a value.
(Personally, I would suggest keeping the names as short as possible!)
